## 작업 관리 원칙

- 새 작업을 시작하면 `현재 상태` 표에 추가한다.
- 작업 중 중요한 조사 결과와 결정 사항을 기록한다.
- 작업을 마치면 상태를 `완료`로 변경하고 결과물 경로를 남긴다.
- 외부 자료를 활용한 문서에는 조사 기준일과 출처를 명시한다.
