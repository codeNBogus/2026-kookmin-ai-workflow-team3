## 파일 규칙

| 파일 | 용도 |
|------|------|
| `progress_<성>.md` | 작업 진행 기록 (필수) |
| 아래 템플릿 | 에이전트 사용 요약 (선택, 개인별로 복사해 사용) |

새 팀원은 [progress-template.md](../../../.cursor/skills/kookmin-team-workflow/progress-template.md)를 복사해 `progress_<성>.md`를 만든다.
